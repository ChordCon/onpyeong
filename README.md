# onpyeong
주의 사항<br />
타인에게 공개된 정보임으로 민감한 정보는 입력하면 안됨.<br />
내용 수정 후 홈페이지 반영까지 시간이 걸릴 수 있음.<br />
<br />
<br />
작성 형식은 아래 내용 참고<br />
온평 성공사례 link : https://github.com/ChordCon/onpyeong/blob/main/case.json <br />
온평 공지사항 link : https://github.com/ChordCon/onpyeong/blob/main/notice.txt <br />
<br />
<br />
성공사례 작성 방식<br />
예시<br />
{<br />
    "id": 1,<br />
    "category": "형사",<br />
    "title": "카메라이용촬영죄 '유죄' 판결을 뒤집고 2심에서 '무죄'를 선고",<br />
    "desc": "카메라이용촬영죄에 대한 1심 '유죄' 판결을 뒤집고 2심에서 '무죄'를 선고받은 사례.",<br />
    "link": "https://blog.naver.com/onpyeong_/224145061687"<br />
}<br />
<br />
: 뒤에 부분만 직접 수정, 앞에 부분 추가 및 제거가 필요 할 경우 연락 바람<br />
다음 내용을 작성할때 이전 내용의 {}에 }에 꼭 쉼표를 추가해야한다. <br />
예시 <br />
[<br />
  {<br />
    "id": 1,<br />
    "category": "형사",<br />
    "title": "카메라이용촬영죄 '유죄' 판결을 뒤집고 2심에서 '무죄'를 선고",<br />
    "desc": "카메라이용촬영죄에 대한 1심 '유죄' 판결을 뒤집고 2심에서 '무죄'를 선고받은 사례.",<br />
    "link": "https://blog.naver.com/onpyeong_/224145061687"<br />
  },<br />
  {<br />
    "id": 2,<br />
    "category": "민사",<br />
    "title": "강간죄 '기소유예' 결정",<br />
    "desc": "강간죄에 대하여 검찰에서 '기소유예' 결정을 받은 사례.",<br />
    "link": "https://blog.naver.com/onpyeong_/224154701660"<br />
  }<br />
]<br />
<br />
id<br />
id에 있는 숫자는 중복되면 안됨 이전에 숫자에서 1 증가 시키면 됨<br />
작성 방법 : 숫자만 입력하고 쉼표 추가 하면 됨 예시) 1,<br />
<br />
category<br />
category는 현재 "전체", "형사", "민사", "가사", "상속", "성범죄", "기타", 로 구성되어있으며 원할시 추가 및 삭제 가능<br />
현재 구성되지 않은 카테고리 입력시 오류가 발생하므로 꼭 추가나 삭제가 필요할시 연락 바람<br />
작성 방법 : 쌍따옴표사이에 카테고리 추가하고 쉼표로 마무리 예시) "형사",<br />
<br />
title<br />
title은 제목으로 위의 예시의 사용된 정도의 길이로 작성 바람<br />
작성 방법 : 쌍따옴표사이에 제목 내용 추가하고 쉼표로 마무리 예시) "카메라이용촬영죄 '유죄' 판결을 뒤집고 2심에서 '무죄'를 선고",<br />
<br />
desc<br />
desc는 간략한 내용으로 위의 예시의 사용된 정도의 길이로 작성 바람 길어도 자동 생략됨<br />
작성 방법 : 쌍따옴표사이에 내용 추가하고 쉼표로 마무리 예시) "카메라이용촬영죄에 대한 1심 '유죄' 판결을 뒤집고 2심에서 '무죄'를 선고받은 사례.",<br />
<br />
link<br />
link는 해당 글의 블로그 주소로 해당 글의 주소창의 주소를 복사해서 붙여넣으면 됨<br />
작성 방법 : 쌍따옴표사이에 링크 주소 추가하고 마무리 예시) "https://blog.naver.com/onpyeong_/224145061687"<br />
주의! 링크는 마지막에 쉼표가 들어가지 않는다.<br />
